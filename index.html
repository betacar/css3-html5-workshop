El DOM

DOM o Document Object Model es un conjunto de utilidades específicamente diseñadas para manipular documentos XML, XHTML y HTML. Técnicamente, DOM es una API de funciones que se pueden utilizar para manipular las páginas XHTML de forma rápida y eficiente.

El árbol generado no sólo representa los contenidos del archivo original (mediante los nodos del árbol) sino que también representa sus relaciones y herencia de estilos, mediante las ramas del árbol que conectan los nodos.

Semántica

  • "Mantenlo simple y estúpido": KISS.
  • "Contenido sobre presentación".
  • Cada tag tiene una definición estándar. Ls tablas (<table>) son para tabular datos, no para diagramación visual.
  • Los títulos no son párrafos con clases, son cabeceras... y tienen orden jerárquico (<h1> ... <h6>).
  • La diagramación visual se controla con CSS, no con HTML. Por ejemplo, no es válido utilizar <br /> para "empujar" un elemento hacia abajo.
  • Para destacar texto se utilizan las etiquetas <em> y <strong>. <b>, <i> y <u> se declaran en desuso.

Selectores

  • Tags (1pt como selector): Afecta a todas las etiquetas del documento.
    • div { ... }: Selecciona todos los <div> del documento.
  • ID (100pts como selector): Se basa en el ID del elemento. Debe ser único por cada documento.
    • #lorem { ... }: Selecciona el elemento con el atributo id="lorem".
  • Clase (10pts como selector): Se basa en la clase del elemento.
    • .azul { ... }: Selecciona todos los elementos con el atributo class="azul".

Selectores

  • Pseudoselector (1pt como selector): Se basa en atributos y "estados" del elemento seleccionado.
    • input[type="button"] { ... }: Selecciona todos los elementos con el atributo type="button".
    • li:first-child { ... }: Selecciona el primer hijo de un conjunto de elementos.
    • a:link|active|hover|visited { ... }: Selecciona los elementos <a>, según su estado.
    • a[href$=".pdf"] { ... }: Selecciona los elementos <a>, donde el valor de href finalice en ".pdf"
  • Selector múltiple: Agrupa un conjunto de selectores.
    • div#container input[type="text"] { ... }
    • ul#menu li.seleccionado:last-child { ... }

Estilos: Propiedades visuales aplicadas a un elemento seleccionado.

  • { background: #c0L0R3 url:("path/a/imagen.jpg") top|center|bottom left|center|right repeat-x|repeat-y|no-repeat; }: Define el fondo de un elemento.
  • { margin:10px; margin: 10px 10px; margin: 10px 10px 10px 10px; margin-[top|right|bottom|left]: 10px; }: Define el margen externo de un elemento.
  • { padding:10px; padding: 10px 10px; padding: 10px 10px 10px 10px; padding-[top|right|bottom|left]: 10px; }: Define el margen interno de un elemento.
  • Fuentes:
